1. 간단한 반복문으로 리스트 각 요소들 출력

{% for data in my_list %} {{data}} {% endfor %}

2. if, else 활용해서 문자열 비교

{% if my_string == input %}

같다

{% else %}

다르다

{% endif %}

3. if, elif, else 사용해보기

{% if my_string|length <= 5 %}

long

{% elif my_string|length >= 10 %}

short

{% else %}

적당

{% endif %}

4.

{% for data in my_list %} {% if data >= 90 %}

A

{% elif data >= 50 %}

B

{% else %}

C

{% endif %} {% endfor %}
